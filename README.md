# Selenium Java Cucumber Automation Framework

Tech Stack:
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven

Framework Features:
- Page Object Model (POM)
- BDD feature files using Gherkin
- Reusable base and utility classes
- Regression test execution

Description:
This repository contains a basic Selenium automation framework
created to demonstrate automation testing skills using Java and Cucumber.
